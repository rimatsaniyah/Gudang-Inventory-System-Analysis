# Warehouse Management System Design

This repository contains the system design for a Warehouse Management System for PT. Medeva Multi Talenta. The project includes various components like business process diagrams, user flow, database design, application architecture, and activity diagrams to illustrate the system's flow and functionality.

## Project Overview
The purpose of this project is to design an application for recording the inventory of goods in a warehouse, from goods receipt to goods dispatch. The design includes:
- Business Process Diagram (BPMN) for tracking goods from arrival to departure.
- User Flow Diagram to outline the user interactions within the system.
- Database Design (ERD) for managing inventory data.
- Application Architecture Diagram to illustrate system components.
- Activity Diagrams for key processes.

## Diagrams
- Business Process Diagram: Outlines the flow from goods receipt to goods dispatch.
- User Flow Diagram: Describes user interaction with the system.
- **Database Design (ERD):** Illustrates the database structure for inventory management.
- **Application Architecture Diagram:** Shows system components and interactions.
- **Activity Diagrams:** Details the specific processes like goods receiving and dispatch.

## Features List
- Detailed list of application features including:
